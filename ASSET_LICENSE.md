# Asset & Audio Declaration

이 프로젝트는 외부 이미지, 외부 폰트, 외부 JavaScript 라이브러리, 외부 음원 파일을 사용하지 않습니다.

- 배경과 UI 그래픽: HTML/CSS로 생성
- 룰렛/슬롯/3D 효과: CSS transforms와 gradients로 생성
- 컨페티/코인/불꽃: 프로젝트 내 JavaScript Canvas 코드로 생성
- 배경음악과 효과음: Web Audio API 오실레이터, 노이즈, 필터를 이용해 실행 시점에 생성
- 파비콘: 문서에 내장한 data URI SVG

따라서 별도 제3자 미디어 저작권 표시가 필요한 에셋은 없습니다.
